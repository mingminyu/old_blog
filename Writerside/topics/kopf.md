# Kopf 教程

<show-structure depth="3"/>

Kopf（Kubernetes Operator Pythonic Framework）是一个用于创建 Kubernetes Operators 的 Python 框架。在 Kubernetes 生态中，Operator 是一种特殊的应用程序，用于自动化对应用程序的管理、创建、配置和修复。
