## 3. AtomGPT

[AtomGPT](https://github.com/mingminyu/AtomGPT) 是我自己尝试使用 NiceGUI 框架仿 ChatGPT 的 WEB UI，整个框架集成了前端 UI 和 FastAPI 后端接口，值得注意的是，该框架暂未提供 LLM NLU 引擎，如果是大模型的应用接口需要单独起一块服务。

通过如下命令来启动 WEB UI:

```Bash
python webui.py
```

> 项目还提供了 Windows 和 Mac 系统的桌面版安装版本。
