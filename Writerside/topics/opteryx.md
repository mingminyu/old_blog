# Opteryx 教程

<show-structure depth="3"/>

opteryx 是一个用于数据处理的 python 库，旨在为大型数据集提供高性能、内存高效的查询和分析功能。opteryx 的目标是让用户能够使用 SQL 语法来处理存储在本地文件系统或数据库中的数据。
