# Conan 教程

<show-structure depth="3"/>

Conan  是 C/C++ 世界中一个崭露头角的包管理器。
它支持自动下载、构建、测试和包装依赖库，让开发者的工作流程更加顺畅。
与  CMake  就如同活塞与汽缸一般，一个负责构建，一个负责管理，各司其职。
更有甚者，当你将眼光投向  vcpkg  或其他  C++ 包管理器时，Conan  以其广泛的平台支持和灵活的解决方案脱颖而出。
