# Py-Spy 教程

<show-structure depth="3"/>

py-spy 是一个强大的 Python 性能分析工具，它可以让你实时查看 Python 程序的运行情况，无需修改代码或重启程序。它就像一个间谍，潜入你的程序内部，揭示其行为和性能瓶颈。
