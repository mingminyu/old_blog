# JsonPickle 教程

<show-structure depth="3"/>

你可能想到了使用 Python 内置的 json 库，但很快就会发现，当对象稍微复杂点，例如有自定义的类实例时，json 库就会束手无策。
而 jsonpickle 就像是一个升级版的魔法师，不仅能处理简单的任务，还能潇洒地转化那些复杂、有深度的对象，为数据的序列化和反序列化提供了极大的便利。
