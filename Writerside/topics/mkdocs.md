# MkDocs 教程

<show-structure depth="3"/>

MkDocs是一个快速、简单且华丽的静态站点生成器，适用于构建项目文档。文档源文件是用 Markdown 编写的，并使用单个 YAML 配置文件进行配置。使用MkDocs，我们将会很方便地给Python项目来生成一个项目文档。
