# Pydub 教程

<show-structure depth="3"/>

Pydub 是一个用于处理音频文件的 Python 库，它提供了简单而强大的接口，用于读取、编辑和保存各种音频文件格式。Pydub 可以帮助你进行音频文件的剪切、合并、转换格式等操作，而且使用方便，适合用于音频处理和编辑的各种场景。
