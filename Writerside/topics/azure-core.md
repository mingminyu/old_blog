# AzureCore 教程

<show-structure depth="3"/>

Azure 是 Microsoft 提供的综合云计算平台，通过 Microsoft 管理的数据中心提供用于构建、部署和管理应用程序和服务的广泛服务。为了与各种 Azure 服务交互，开发人员可以使用 Python 中的 azure-core 模块，该模块提供了有效使用 Azure 服务的核心功能和实用程序。
