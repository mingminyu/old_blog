# Daphne 教程

<show-structure depth="3"/>

Daphne由Django Software Foundation开发，旨在提供一个简单且高效的ASGI服务器，用于运行异步的Python Web应用。Daphne的设计目标是易于安装和使用，同时提供良好的性能和稳定性。它支持WebSocket和HTTP/2等现代Web技术，使得开发者能够构建高性能的实时Web应用。
