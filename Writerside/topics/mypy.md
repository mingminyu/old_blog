# MyPy 教程

<show-structure depth="3"/>

Mypy 是一个静态类型检查器，用于 Python 语言。它允许开发者为 Python 代码添加类型注解，然后使用这些注解来检查代码的类型错误。Mypy 的目标是在不改变 Python 动态类型特性的前提下，提供类型检查的功能，帮助开发者发现潜在的类型错误，提高代码的可靠性和可维护性。
