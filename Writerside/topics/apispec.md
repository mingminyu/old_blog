# Apispec 教程

<show-structure depth="3"/>

在构建 RESTful API 的海洋中，文档就像一座灯塔，指引着开发者安全地航行于无尽的接口与功能之间。
那么，如何高效地构建这些指路明灯呢？
我们不禁会想，要是有一个工具能自动帮我们完成文档的编织，甚至还能让不同的框架共用一套标准，那该多好。apispec，一个轻松实现 API 文档生成与验证的 Python 神器，就是为此而生。
