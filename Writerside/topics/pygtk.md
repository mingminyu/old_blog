# PyGTK 教程

<show-structure depth="3"/>

pygtk 是一个用于创建图形用户界面（GUI）的 Python 库，它提供了对 GTK+ 库的绑定。GTK+ 是一个广泛使用的开源工具包，用于创建跨平台的桌面环境和应用程序。pygtk 允许 Python 开发者利用 GTK+ 的强大功能来设计和构建复杂的用户界面。
