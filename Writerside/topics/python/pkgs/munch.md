# Munch 教程

<show-structure depth="3"/>

munch 是一个轻量级的 Python 类库，它给予了 Python 的字典一个特殊的能力：
使用点号(.)来访问字典中的元素。这好比是给字典穿上了一个允许它以对象属性的方式出现的外衣，同时，这个外衣设计得既合身又舒适，不会影响它原有的功能。
