# Luigi 教程

<show-structure depth="3"/>

Python Luigi 是一个用于构建复杂数据管道的 Python 库。它的设计灵感来自于 Google 的 MapReduce 和 Apache Hadoop 项目。Luigi 的核心思想是将数据处理流程划分为多个任务，并定义这些任务之间的依赖关系，从而实现数据流的自动化管理和调度。
