# Casbin 教程

<show-structure depth="3"/>

Casbin是一个强大的、高效的开源访问控制库，它支持多种访问控制模型，如RBAC（基于角色的访问控制）、ABAC（基于属性的访问控制）等。Casbin的Python接口称为pycasbin，允许你在Python项目中使用Casbin进行权限管理。本教程将介绍如何使用pycasbin模块来设置和管理访问控制策略。
