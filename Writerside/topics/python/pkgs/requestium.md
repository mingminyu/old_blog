# Requestium 教程

<show-structure depth="2"/>

在 Python 编程中，处理网络请求是一个常见的任务，特别是做爬虫采集数据。最受欢迎的是 Requests 和 Selenium。而Requestium 结合了这两个库优点的工具，它可以让你在一个统一的接口中使用 Requests 的简便性和 Selenium 的强大功能。
