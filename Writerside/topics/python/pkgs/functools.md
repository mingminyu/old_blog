# Functools 教程

<show-structure depth="3"/>

functools是Python标准库中的一个模块，它提供了一系列用于高阶函数：即那些作用于或返回其他函数的函数。这些工具主要用于函数式编程风格，其中包括用于创建函数包装器的装饰器。
