# Openpyxl 教程

<show-structure depth="2"/>

openpyxl 是一个用于读取和写入 Excel 2010 及以上版本的 xlsx/xlsm/xltx/xltm 文件的 Python 库。这个库提供了与 Excel 工作簿进行交互的功能，非常适合进行自动化或批量处理 Excel 数据。
