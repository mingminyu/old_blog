# Zappa 教程


<show-structure depth="2"/>

在当今云计算时代，Serverless 架构正逐渐成为开发者的首选。Python Zappa 库作为一个优秀的工具，使得开发者可以轻松地将 Python 应用部署到 Serverless 平台上。本文将深入探讨 Zappa 库的功能、用法以及如何利用它来构建强大的 Serverless 应用。
