# DearPyGui 教程

<show-structure depth="2"/>

Python DearPyGui是一个基于Python的GUI库，它提供了一种简单易用的方式来创建跨平台的图形用户界面（GUI）。它是一个轻量级的库，使用起来非常方便，同时还具有高度的可定制性和灵活性。在本文中，我们将介绍Python DearPyGui的一些特点和优势，以及如何使用它来创建GUI应用程序。
