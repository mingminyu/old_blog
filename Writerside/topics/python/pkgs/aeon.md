# Aeon 教程

<show-structure depth="2"/>

aeon 是一个与 scikit-learn 兼容的工具包，用于执行预测、分类和聚类等时间序列任务。它提供了广泛的时间序列算法，包括最新进展，并使用 numba 高效实现时间序列算法。
