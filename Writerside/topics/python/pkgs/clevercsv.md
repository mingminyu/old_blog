# CleverCSV 教程

<show-structure depth="2"/>

CleverCSV 是一个基于 Python 的库，旨在提供比标准库 csv 更智能和灵活的方法来处理 CSV 文件。该库使用机器学习算法来探测 CSV 文件的正确拨号结构，从而解决不同 CSV 文件格式导致的读取问题。它尤其适用于处理具有复杂结构或非标准分隔符的CSV文件。
