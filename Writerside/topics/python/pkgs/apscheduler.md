# ApScheduler 教程

<show-structure depth="3"/>

apscheduler 是一个开源的 Python 库，用于在应用程序中添加作业调度功能。它非常灵活，允许你计划任务的执行，这些任务可以是一次性的、固定间隔的或者基于复杂调度（如在特定时间执行）的。APScheduler 广泛用于构建复杂的调度需求应用程序，如定时任务、后台作业处理等。
