# Sumy 教程

<show-structure depth="2"/>


Sumy库是一个用于自动文本摘要的Python库，支持多种摘要算法，如TextRank、LSA、Luhn、Edmundson等。这些算法允许我们从文本中提取出最重要的句子或生成更简洁的摘要，帮助我们更高效地理解大量文本。
