# ChineseCalendar 教程

<show-structure depth="3"/>

工作与学习中，有个场景需要判断指定日期是不是工作日，例如自动化办公工作日自动发邮件，量化交易判断是否为交易日等等， 因此我们需要能准确判断给的日期是不是工作日，Python中一般使用模块chinesecalendar来解决这个问题。
