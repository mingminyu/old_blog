# Mlxtend 教程

<show-structure depth="2"/>

mlxtend（Machine Learning Extensions）是一个流行的 Python 库，其中包含「用于数据科学和机器学习任务」的有用工具。它不是替代像 scikit-learn 这样的主流机器学习库，而是作为补充，提供一些在这些库中不常见的功能和工具。
