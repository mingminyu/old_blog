# Yarf 教程

<show-structure depth="3"/>


YAPF（Yet Another Python Formatter）是由Google开发的Python代码格式化工具。

与其他代码格式化工具相比，YAPF的最大特点是它的设计理念：简单、易用和可配置。

无论你的代码风格如何，YAPF都可以帮助你将其格式化为一种更加整洁、一致的风格。
