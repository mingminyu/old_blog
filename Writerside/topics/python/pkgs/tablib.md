# Tablib 教程

<show-structure depth="2"/>

Tablib 是一个Python库，用于创建、操作和导出多种表格格式（如CSV、XLSX、JSON等）。它的设计目标是简单易用，同时保持高性能。Tablib 支持多种数据类型，如字符串、数字、日期和布尔值，并提供了丰富的方法来操作这些数据，如添加、删除、修改行和列等。这使得Tablib 成为数据处理和分析的有力工具，特别是在需要多种数据输出格式的场合。
