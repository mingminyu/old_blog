# Kornia 教程

<show-structure depth="2"/>

Kornia 是一个开源的 Python 库，专注于提供深度学习中的几何变换和计算机视觉相关的功能。它建立在 PyTorch 之上，为研究者和开发者提供了一系列的算法和工具，用于处理图像、视频和点云数据。Kornia 的设计目标是简化复杂的几何变换和视觉任务，同时保持高性能和易用性
