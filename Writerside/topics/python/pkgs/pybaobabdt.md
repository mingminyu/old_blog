# Pybaobabdt 教程

<show-structure depth="2"/>

pybaobabdt 是一个基于 Python 的决策树（Decision Tree）和随机森林（Random Forest）可视化工具。这个库的主要目的是提供一个直观、易于理解的方式来展示和分析决策树和随机森林模型的结构。
