# Signal 教程

<show-structure depth="3"/>

Python的signal模块允许程序员发送Unix信号到进程，以及注册信号处理器来捕获这些信号。Unix信号是一种软件中断，用于通知进程有某种事件发生了。例如，当你尝试终止一个进程时，系统会发送一个SIGTERM信号给该进程。
