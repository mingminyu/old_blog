# Klib 教程

<show-structure depth="2"/>

klib提供了一系列非常易于应用的函数，具有合理的默认值，几乎可以用在任何DataFrame上，用于评估数据质量、获得灵感、执行数据清洗和可视化，从而更轻便、更高效的处理Python Pandas DataFrame数据。
