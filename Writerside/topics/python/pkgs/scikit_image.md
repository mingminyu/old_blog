# ScikitImage 教程

<show-structure depth="2"/>

scikit-image 是一个开源的Python包，它为图像处理提供了丰富的算法和工具箱。无论你是想要改变图片大小、过滤噪声、检测边缘还是进行复杂的图像分割，scikit-image 都能帮到你。它建立在 ciPy 生态之上，意味着它非常适合科学计算中的图像处理任务。
