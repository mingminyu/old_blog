# Interpret 教程

<show-structure depth="2"/>

InterpretML 是一个开源的 Python 库，用于解释机器学习模型的预测结果。它提供了多种解释技术，帮助用户理解模型的内部工作机制，并识别出影响模型预测的关键特征。
