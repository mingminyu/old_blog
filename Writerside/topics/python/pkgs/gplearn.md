# GpLearn 教程

<show-structure depth="2"/>

gplearn 是一个用于执行遗传编程（Genetic Programming, GP）的 Python 库，它建立在 scikit-learn 的接口之上，提供了一种灵活、高效的方法来发现数据中的模式和关系。虽然遗传编程 (GP) 可用于执行非常广泛的任务，但 gplearn 被有意限制为解决符号回归问题。
