# Html2Text 教程

<show-structure depth="2"/>

html2text是一个Python库，它能够将HTML文档转换为纯文本格式。这意味着所有的HTML标签、样式和脚本都会被移除，只留下干净、易读的文本内容。对于不熟悉HTML结构的小白用户来说，html2text提供了一个简单直接的方式来获取网页中的文本信息。
