# Faker 教程

<show-structure depth="2"/>

你是否曾在设计数据库时，手捏数据填充表格，以便模拟真实世界的环境？
或者当你在设计一个用户界面而需要展示预览时，却发现自己缺乏内容上的灵感？
在这些场景中，人们往往需要大量、多样化，仿佛真实存在的数据——这就是伪数据的用武之地。
Python  的 Faker 库正是你在此时需要的帮手，它能够轻松生成各类伪数据，让你的测试和设计工作简单又有趣。
