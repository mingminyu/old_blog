# ShortUUID 教程

<show-structure depth="2"/>

想象一下，你正在设计一个系统，需要生成大量的唯一 ID，作为数据库条目标识或在网络上进行共享。你不仅希望这些  ID  具有全局唯一性，而且还要它们简短易传输和记忆。

shortuuid 就是你需要的解决方案，它能够为你生成简洁而独特的 UUID。
