# Diagrams 教程

<show-structure depth="2"/>


Diagrams是一个通过Python代码生成云系统架构图的库。这个库特别适合云架构师和开发人员，他们可以使用Diagrams来绘制出云服务的架构图，而无需依赖传统的绘图工具。Diagrams支持多个云服务提供商的图标，如AWS、Google Cloud Platform、阿里、谷歌、Azure等，使得生成的架构图既美观又专业。
