# Prophet 教程

<show-structure depth="2"/>

Prophet是由Facebook开发的一种时间序列预测库。它旨在简化时间序列数据的建模和预测，使得即使对于没有专业领域知识的用户来说，也能够轻松地通过几行代码进行时间序列的分析和预测。
