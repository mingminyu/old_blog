# Mistune 教程

<show-structure depth="2"/>

Mistune是一个用Python编写的快速、轻量级的Markdown解析器。它支持标准的GitHub Flavored Markdown（GFM）以及一些额外的扩展语法。Mistune的设计目标是尽可能地接近Markdown的原始实现，同时提供足够的扩展性以满足不同的需求。
