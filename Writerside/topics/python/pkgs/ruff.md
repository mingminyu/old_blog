# Ruff 教程

Ruff，它是基于 Rust 编写的高性能 Python 代码分析工具（即 Linter），用于检查代码中的语法错误、编码规范问题、潜在的逻辑问题和代码质量问题等，可以提供实时反馈和自动修复建议。
