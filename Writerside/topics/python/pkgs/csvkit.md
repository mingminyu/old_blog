# CSVKit 教程

<show-structure depth="2"/>

在数据科学和数据分析的领域中，CSV（逗号分隔值）格式因其简洁性和广泛的兼容性而广受欢迎。CSV 文件通常包含表格数据，可以轻松地被大多数电子表格软件和数据库应用程序读取和处理。
