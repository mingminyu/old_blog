# Chameleon 教程

<show-structure depth="3"/>

Chameleon是一个高性能的模板引擎，适用于Python编程语言。它最初是为了服务于Zope和Pyramid这两个Web框架而设计的，但它同样可以在任何Python项目中独立使用。Chameleon的语法灵感来源于ZPT（Zope Page Templates），旨在提供一个清晰、简洁的模板语言。
