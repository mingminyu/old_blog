# Autoviz 教程

<show-structure depth="2"/>

Autoviz是一个用于自动可视化的Python库，它可以根据给定的数据集自动生成多种类型的图表。它的设计目标是使数据探索和分析过程更加高效和便捷，无需手动编写大量的可视化代码。Autoviz基于Matplotlib和Seaborn等流行的可视化库，提供了一种简单且直观的方式来探索和展示数据集的特征和关系。
