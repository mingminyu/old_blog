# Tomli 教程

<show-structure depth="3"/>

TOML 是一种强调可读性和简单性的配置文件格式。它被设计为易于人类和计算机编写和理解。 TOML 文件使用键值对方法，类似于 INI 文件，但增加了一些功能，例如支持嵌套数据结构和整数、浮点数和字符串等数据类型。
