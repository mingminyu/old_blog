# HandCalcs 教程


HandCalcs是一个非常实用的Python模块，可以帮助开发者轻松地从Python代码中生成LaTeX报告。使用HandCalcs，你只需编写最少的代码，就能自动生成符合LaTeX格式的数学方程。
