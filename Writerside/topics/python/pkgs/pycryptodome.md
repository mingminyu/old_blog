# PyCryptodome 教程

<show-structure depth="2"/>

工作中，设计api接口时，为了数据安全一般都会设计签名以便于服务提供方认证调用方，例如我工作中调用淘宝，微信，京东等都会涉及签名认证，而日常中常见的会使用RSA这种非对称加密算法。Python里一般通过ras模块来生成或加载秘钥，通过pycryptodome进行加密与验签。
