# Pymeter 教程

<show-structure depth="2"/>

JMeter 是一个流行的性能测试工具，用于测试 Web 应用程序的性能和负载。它通常与 GUI 一起使用，但如果您想在自动化测试中集成 JMeter，或者以编程方式创建和运行测试计划，那么 pymeter 库将是一个强大的工具。本文将介绍如何使用 Python 和 pymeter 库进行 JMeter 测试的实际操作。
