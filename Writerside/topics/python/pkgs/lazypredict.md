# LazyPredict 教程

<show-structure depth="2"/>

lazypredict 是一个开源的 Python 库，旨在简化机器学习模型的构建和评估过程。使用 lazypredict 无需太多代码即可帮助构建大量基本模型，并有助于了解哪些模型在无需任何参数调整的情况下效果更好。
