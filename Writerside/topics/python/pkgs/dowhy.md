# Dowhy 教程

<show-structure depth="2"/>

dowhy 是一个开源的 Python 库，旨在简化因果推断的过程，特别是针对那些希望从数据中估计因果效应的研究者和数据科学家。通过它可以轻松地定义因果模型、识别因果效应、估计这些效应以及进行健壮性检验。它建立在 Pandas、NumPy 和 scikit-learn 等流行库之上，提供了一个直观且强大的工具，用于根据观测数据估计因果效应。
