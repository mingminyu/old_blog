# Pdoc 教程

<show-structure depth="2"/>

pdoc 是一个轻量级的库，专注于为  Python  项目生成  API  文档。

它通过扫描指定的模块和包，自动提取文档字符串，快速转化为易于浏览的  HTML  文档。这项技术尤其适用于需要快速生成文档的现代  Python  项目。
