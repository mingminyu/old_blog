# Filelock 教程

<show-structure depth="3"/>

filelock 是一个纯 Python 写成的库，它提供了一种简易实用的方式来实现跨平台的文件锁功能。
你可能不会经常听到它的名字，因为它默默无闻地在后台为你守护着文件的安全
