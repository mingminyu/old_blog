# Pandarallel 教程

<show-structure depth="2"/>

Pandas是非常著名的Python数据分析库，几乎是所有做大数据分析、机器学习、人工智能方向的必备技能；同时由于Pandas对时间序列操作的支持特别好，所以在量化交易的研发工作中经常应用。

而Pandarallel正是一个简单而有效的Pandas并行化加速工具，仅通过一行代码的更改就可以在所有平台上利用所有可用CPU并行Pandas操作。
