# GeoPandas 教程

<show-structure depth="2"/>

Geopandas是一个开源项目，它扩展了Pandas库的功能，使得处理和分析地理空间数据变得轻而易举。简单来说，如果你熟悉Pandas的DataFrame结构，那么你会发现Geopandas提供了一个类似的GeoDataFrame，它能够轻松地处理地理空间数据。
