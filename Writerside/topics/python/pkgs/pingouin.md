# Pingouin 教程

<show-structure depth="2"/>


Pingouin(https://pingouin-stats.org/index.html) 是一个开源的Python库，支持各种假设检验和统计模型  。
该库包括许多测试，例如ANOVA、t检验、卡方检验、克鲁斯卡尔-沃利斯检验(Kruskal-Wallis)、曼-惠特尼检验(Mann-Whitney)、威尔科克森符号秩(Wilcoxon signed-rank)等。
