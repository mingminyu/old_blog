# Altair 教程

<show-structure depth="2"/>


altair 是一个基于 python 的声明式统计可视化库，它建立在 Vega 和 Vega-Lite 可视化语法之上。altair 的设计目的是让用户能够以简洁、直观的方式构建各种统计图表，同时保持图表的灵活性和可扩展性。通过使用 altair，用户可以轻松地创建和探索数据的可视化表示，从而更有效地理解数据和传达信息。
