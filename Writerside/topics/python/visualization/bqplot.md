# BqPlot 教程

<show-structure depth="2"/>

Bqplot 是一个用于数据可视化的高效库，帮助开发者创建交互式和自定义图表。通过 Bqplot，您可以轻松地将数据转换为美观且直观的可视化图表，从而更好地展示和分析数据。
