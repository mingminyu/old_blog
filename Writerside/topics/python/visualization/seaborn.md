# Seaborn 教程

<show-structure depth="2"/>

Seaborn是一个基于Python的数据可视化库，Seaborn提供了许多用于绘制统计图形的高级界面，在Matplotlib的基础上，Seaborn进行了更高级的封装，使得作图更加方便快捷。即便是没有什么基础的人，也能通过极简的代码，做出具有分析价值而又十分专业的图形。
