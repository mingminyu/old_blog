# Solara 教程

<show-structure depth="2"/>

Solara是一个开源的Python库，旨在简化交云端数据处理和交互式Web应用的开发过程。
它基于流行的数据科学库如Pandas和Plotly，提供了一种高效、直观的方式来创建数据驱动的Web应用。
