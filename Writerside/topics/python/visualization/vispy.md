# VisPy 教程

<show-structure depth="3"/>

根据 VisPy 官网的内容，VisPy 是一个高性能的交互式 2D/3D 数据可视化库。
它利用现代图形处理单元（GPUs）通过 OpenGL 库的计算能力来显示大型数据集。
