# Datashader 教程

<show-structure depth="3"/>


与传统的数据可视化库（如  matplotlib  或 bokeh）不同，Datashader  不只是简单地绘制点、线和柱，而是使用一种称为 **"像素级渲染"** 的技术来处理大规模的数据集。
