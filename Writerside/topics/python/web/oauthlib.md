# OAuthlib 教程

<show-structure depth="3"/>

OAuthLib是一个专为Python设计的库，提供了一个高级接口，用于构建OAuth1和OAuth2客户端和服务端。简而言之，它是实现OAuth授权流程的利器。OAuthLib的设计哲学是简单、安全和灵活，使得它成为Python开发者处理OAuth协议的首选库。
