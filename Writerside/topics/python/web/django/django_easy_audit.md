# DjangoEasyAudit 教程

<show-structure depth="2"/>

Django Easy Audit是Django审计日志应用程序， 使用Django 信号来监听项目中发生的事件，例如用户创建、更新或删除时进行追踪记录，适用于项目中每个应用的每个模型。
