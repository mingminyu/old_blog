# JustPy 教程

<show-structure depth="3"/>

JustPy是一个面向对象的高级Python Web框架，它不需要前端编程。它的设计目标是使Web开发更加简单和快速。JustPy提供了一种简单而强大的方法来创建动态的Web应用程序，而无需编写大量的JavaScript代码。
