# FastAPI-React 教程

<show-structure depth="3"/>

FastAPI-React 是一个 Cookiecutter 模板，旨在简化使用 FastAPI 和 React 构建全栈 Web 应用程序的过程。它整合了 PostgreSQL、SQLAlchemy 和 Docker，提供了一个健壮且可扩展的开发环境，是学习借鉴全栈Web应用程序开发的不二之选。
