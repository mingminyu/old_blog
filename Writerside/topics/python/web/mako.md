# Mako 教程

<show-structure depth="3"/>

Mako是一个用Python编写的模板库，它可以将Python表达式和控制结构嵌入到HTML或其他类型的文档中。Mako的设计旨在快速且易于使用，同时提供足够的灵活性来满足复杂的渲染需求。
