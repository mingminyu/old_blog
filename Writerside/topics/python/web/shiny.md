# Shiny 教程

<show-structure depth="3"/>

Python Shiny 是一个用于创建交互式 Web 应用程序的开源库，它基于 Flask 和 React 技术栈，提供了丰富的功能和工具，使得用户可以轻松地构建和部署交互式应用。本文将深入介绍 Shiny 库的各项功能和使用方法，并提供丰富的示例代码，以帮助大家更好地理解和应用这个强大的工具。
