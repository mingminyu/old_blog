# Cherrypy 教程

<show-structure depth="2"/>

CherryPy 是一个轻量级的 Python Web 框架，它使开发 Web 应用变得简单而愉快。无论是构建小型项目还是大型应用，CherryPy 提供了一种灵活且易于使用的方式来处理 Web 请求和构建 Web 页面。本文将介绍 CherryPy 的基本概念、安装、配置以及如何创建一个简单的 Web 应用，并提供丰富的示例代码。
