# Mlforecast 教程

<show-structure depth="3"/>

mlforecast是一个基于Python的机器学习预测库，专为时间序列预测而设计。它结合了传统统计方法和现代机器学习技术，提供了一个简单、高效的解决方案来进行大规模时间序列预测。
