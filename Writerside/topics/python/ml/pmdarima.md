# Pmdarima 教程

<show-structure depth="2"/>


pmdarima 是一个开源的 Python 库，用于时间序列预测。Pmdarima 的 auto_arima 函数在构建 ARIMA 模型时非常有用，因为它可以帮助我们识别最佳的 p、d、q 参数并返回拟合的 ARIMA 模型。
