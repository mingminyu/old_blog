# FeatureEngine 教程

<show-structure depth="2"/>

Feature Engine 是一个开源的 Python 包，专为特征工程而设计。
它提供了一系列工具，帮助数据科学家和机器学习工程师高效地进行特征选择、特征转换、特征创建等任务。
