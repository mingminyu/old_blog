# Scikit-Optimize 教程

<show-structure depth="3"/>

Scikit-Optimize（skopt）是一个用于模型超参数优化的Python库，它提供了一个简单而强大的接口，用于最小化或最大化一个目标函数。skopt使用机器学习模型来预测超参数的最佳组合，从而减少需要评估的参数组合数量，这对于需要大量计算资源的模型尤其有用。
