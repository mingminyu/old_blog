# IRedis 教程

<show-structure depth="3"/>

IRedis是一个基于Python的Redis命令行工具，它提供了一个增强的交互式界面，使得与Redis的交互更加直观和高效。IRedis支持自动完成和语法高亮，能够让你快速地执行Redis命令，查看结果。此外，它还提供了一些高级功能，如命令历史记录、响应时间显示等，使得操作Redis变得更加轻松。
