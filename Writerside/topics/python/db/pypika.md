# Pypika 教程

<show-structure depth="3"/>

Pypika 是一个设计用于动态生成 SQL 查询的 Python 库。它的主要目标是简化构建复杂查询的过程，同时保持可读性。它提供了流畅的 API，允许用户以 Pythonic 的方式使用方法链的语法创建 SQL 查询
