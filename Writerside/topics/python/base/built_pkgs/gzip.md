# Gzip 教程

<show-structure depth="3"/>

gzip模块在Python中用于读取和写入gzip压缩和解压缩的文件。gzip是一种使用LZ77算法和Huffman编码的数据压缩格式。通过使用gzip，你可以显著减少文件的大小，从而节省存储空间，并在传输数据时减少带宽使用。
