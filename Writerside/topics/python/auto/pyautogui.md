# PyAutoGUI 教程

<show-structure depth="2"/>

PyAutoGUI是一个强大的Python库，它允许你的脚本控制鼠标和键盘，自动执行各种任务。
这个库特别适合于那些需要自动化日常重复办公任务的场景，比如自动填写表格、自动点击界面、自动移动文件等。
