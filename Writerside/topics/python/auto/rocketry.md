# Rocketry 教程

<show-structure depth="2"/>

Rocketry 是一个现代Python任务调度库，它允许开发者以极简的方式定义和执行定时任务。这个库特别适合需要在Python环境中自动化执行重复任务的场景，如数据抓取、定时报告、系统维护操作等。Rocketry 的设计哲学强调简洁性和灵活性，使得任务调度变得既容易又强大。
