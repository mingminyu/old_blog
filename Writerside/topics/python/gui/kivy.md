# Kivy 教程

<show-structure depth="3"/>


Kivy是一个开源的Python库，用于开发多触摸应用程序。它适用于Android、iOS、Linux、OS X和Windows平台。这使得Kivy成为跨平台移动应用和桌面应用开发的理想选择。在本文中，我们将深入了解Kivy的核心特性、如何设置环境、编写简单的Hello World程序，以及Kivy在实际项目中的应用。
