# wxGlade 教程

<show-structure depth="2"/>

wxGlade，是一个强大的GUI构建工具，它使用Python编写，为wxWidgets / wxPython提供了全面的支持。无论您是初学者还是有经验的开发者，wxGlade都能帮助您快速创建出精美的图形用户界面。本文将详细介绍wxGlade的特点和使用方法，帮助您更好地了解和使用这个爆款工具
