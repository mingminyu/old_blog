# Spei-Python 教程

<show-structure depth="3"/>

spei-python 是一个基于 Python 的库，用于处理德国银行账户的支付信息交换 (SPEI) 格式。它提供了一组功能强大的工具，使你能够解析和生成 SPEI 文件，以及对其进行验证、读取和操作。
