# Cuisine 教程

<show-structure depth="3"/>

Cuisine 是一个 Python 日期和时间处理库，它的目标是让日期和时间的操作变得更加简单和直观。Cuisine 提供了一系列丰富的功能，如日期和时间的加减、格式化、计算等，可以帮助开发者轻松地处理复杂的日期和时间问题。此外，Cuisine 还提供了一些实用的工具，如周期计算、节假日计算等，可以帮助开发者更好地管理时间和任务。
