# OpenCV 教程

<show-structure depth="3"/>

OpenCV（开源计算机视觉库，Open Source Computer Vision Library）是一个开源的计算机视觉和机器学习软件库。它包含了许多用于实时计算机视觉的优化算法。OpenCV 的主要目标是为计算机视觉应用程序提供一个通用的基础设施，加速计算机视觉应用程序的开发和使用。
