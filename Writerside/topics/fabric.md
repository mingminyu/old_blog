# Fabric 教程

<show-structure depth="3"/>

Python Fabric库是一个用于简化远程部署和系统管理的工具库。它提供了一组简洁而强大的函数和工具，可以帮助开发者轻松地在多台远程主机上执行命令、上传文件、下载文件等操作，从而实现自动化部署和管理。
