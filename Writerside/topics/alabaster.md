# Alabaster 教程

<show-structure depth="3"/>

Alabaster  是一个为  Sphinx  文档系统设计的轻量级主题，以其清洁、响应式和可配置的特性而闻名。
与其他复杂繁杂的  Sphinx  主题相比，Alabaster  更加注重于提供一个简洁现代的用户体验。
最初，它是作为一个第三方主题发布的，直到  Sphinx 1.3 版本，Alabaster  成为了  Sphinx  的默认主题，并被直接包含在内。
