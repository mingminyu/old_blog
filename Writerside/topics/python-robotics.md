# PythonRobotics 教程

<show-structure depth="3"/>

PythonRobotics 是一个开源项目，它包含了一系列的机器人算法和示例，这些算法和示例用 Python 语言编写。该项目旨在提供机器人学中常见问题的解决方案，包括路径规划、运动规划、传感器数据融合、定位、建图（SLAM）等。
