# Napalm 教程

<show-structure depth="3"/>

NAPALM (Network Automation and Programmability Abstraction Layer with Multivendor support) 是一个 Python 库，用于简化网络设备的自动化管理。它提供了一个统一的 API，可以与来自不同供应商的网络设备进行交互，从而实现跨平台的网络自动化。
