# JmesPath 教程

<show-structure depth="3"/>

JmesPath 是一种查询语言，用于分析和操作 JSON 数据。它允许用户通过表达式来选择 JSON 文档中的元素，类似于 XPath 在 XML 中的作用。JmesPath 被设计为非常简洁和易于使用，同时非常强大，能够表达复杂的查询。
