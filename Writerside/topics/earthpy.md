# Earthpy 教程

<show-structure depth="3"/>

EarthPy是一个 python 软件包，它使使用开源工具绘制和处理空间栅格和矢量数据变得更加容易。Earthpy 依赖于侧重于矢量数据的 geopandas 和便于输入和输出光栅数据文件的rasterio，你还可以使用 matplotlib 进行绘图操作。
