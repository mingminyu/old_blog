# Attrs 教程

<show-structure depth="3"/>

Attrs 是一个 Python 库，它提供了一种简洁而强大的方式来定义类，并自动生成相关的特殊方法（如 `__init__`、`__repr__`、`__eq__` 等）。这使得编写和维护 Python 类变得更加简单和高效。
