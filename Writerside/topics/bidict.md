# BiDict 教程

<show-structure depth="2"/>

在 Python 中，字典（dict）是一种常见的数据结构，用于存储键值对。然而，在某些情况下，需要一种能够同时支持正向和反向查找的数据结构，这就是双向字典。Python 中的 bidict 库就提供了这样的功能。本文将深入介绍 Python 中的 bidict 库，包括其特性、用法以及实际应用场景，并提供丰富的示例代码。
