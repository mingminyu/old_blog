# Theano 教程

<show-structure depth="3"/>

Theano 是一个 Python 库，用于定义、优化和评估涉及多维数组的数学表达式。它曾经是深度学习领域的重要工具，为 TensorFlow 和 PyTorch 等现代深度学习框架奠定了基础。
