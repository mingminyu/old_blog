# MpMath 教程

<show-structure depth="3"/>


mpmath  是一个开源的 Python 库，它提供了用于处理任意精度的浮点数(arbitrary-precision floating-point numbers)的功能。这个库广泛用于科学计算、金融、物理学以及工程领域的数值计算中。mpmath 支持许多数学函数，包括基本算术运算、三角函数、特殊函数以及线性代数运算等。
