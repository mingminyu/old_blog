# Kangas 教程

<show-structure depth="3"/>

Kangas 是一个用于探索、分析和可视化大规模多媒体数据的工具，提供了一个简单的 Python API 用于记录大量数据表，以及一个直观的视觉界面，用于对数据集执行复杂查询。
