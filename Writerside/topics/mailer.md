# Mailer 教程

<show-structure depth="3"/>

在编程和软件开发中，"mailer" 通常指的是用于发送电子邮件的软件或服务。在 Python 中，有几个库可以用来实现邮件发送的功能，其中最著名的是 smtplib 和 email 模块，这些是 Python 标准库的一部分，专门用于构建和发送电子邮件。
