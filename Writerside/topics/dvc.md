# DVC 教程

<show-structure depth="3"/>

DVC 是一个开源工具，它使用简单的命令行界面，帮助数据科学家和机器学习工程师管理他们的数据和模型。
DVC 也提供了 VS Code 扩展，用于帮助开发可重复的机器学习项目。
