# Agate 教程

<show-structure depth="3"/>

Agate  不仅仅是一个  Python  数据分析库，它是数据探索和验证过程中的好伙伴。
Agate  的设计宗旨在于尽可能简化与  CSV  文件或任何可转化为表格数据的交互，它的易用性使得它成为不少数据分析师的利器。
