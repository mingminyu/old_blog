# Python-Patterns 教程

<show-structure depth="3"/>

Python Patterns 库是一个开源的 Python 库，旨在提供各种常见的设计模式实现。它包含了大量的设计模式示例代码，涵盖了创建型、结构型和行为型设计模式，如工厂模式、单例模式、装饰器模式、观察者模式等。
