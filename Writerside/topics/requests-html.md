# Requests-Html 教程

<show-structure depth="3"/>


Requests-HTML是为现代网页设计的，旨在使网页抓取变得简单。它集成了Requests库的所有功能，并且添加了HTML解析功能，让你轻松处理JavaScript渲染的内容。
