# Model_Mommy 教程

<show-structure depth="3"/>

model_mommy 是一个曾经用于 Django 应用程序的 Python 库，它提供了一种快速创建测试数据的方法。这个库允许开发者生成模型实例，用于测试 Django 应用程序的不同方面，确保代码的正确性和稳定性。
