# Progress 教程

<show-structure depth="3"/>

progress 是一个用于显示进度条的 Python 库，它可以在终端或命令行界面中显示任务的完成进度。进度条可以提供即时的视觉反馈，帮助用户了解长时间运行任务的执行情况。
