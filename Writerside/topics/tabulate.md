# Tabulate 教程

<show-structure depth="3"/>

Tabulate  是一个用于生成简洁的 ASCII 表格的 Python 库。它可以将数据列表、字典列表或 Pandas DataFrame 转换成一个美观的表格，以便于打印或展示。tabulate 支持多种表格格式，包括简单的网格、Markdown、HTML 和 LaTeX。
