# Pylibmc 教程

<show-structure depth="3"/>

Memcached 是一个广泛使用的高性能内存对象缓存系统，用于减少数据库负载、加快数据访问速度。Pylibmc 是一个 Python 客户端库，它提供了与 Memcached 服务器交互的接口，使得 Python 应用程序可以轻松地利用 Memcached 作为其缓存解决方案。
