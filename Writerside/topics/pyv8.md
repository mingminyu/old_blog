# Pyv8 教程

<show-structure depth="3"/>

`PyV8` 是一个 Python 扩展，它允许你在 Python 中嵌入 Google 的 V8 JavaScript 引擎。这意味着你可以使用 Python 来执行 JavaScript 代码，而无需依赖于浏览器环境。
