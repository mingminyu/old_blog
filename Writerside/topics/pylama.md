# Pylama 教程

<show-structure depth="3"/>

pylama 是一个用于检查 Python 代码风格和潜在错误的工具。它通过整合多个流行的 linters（如 pylint、pyflakes）和格式检查器（如 pep8），为开发者提供了一个方便且全面的代码质量检查方案。使用 pylama 可以帮助开发者快速定位代码中的问题，并遵循一致的代码风格。
