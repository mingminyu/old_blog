# BeeWare 教程

<show-structure depth="3"/>

BeeWare 是一个开源项目，旨在帮助开发者使用 Python 创建原生应用程序，覆盖了移动、桌面和 Web 平台。BeeWare 通过提供一系列工具和库，使得开发者可以利用 Python 的简洁和强大来构建跨平台的应用程序，而无需学习其他语言或工具。
