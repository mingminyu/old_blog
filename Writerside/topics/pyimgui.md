# Pyimgui 教程

<show-structure depth="3"/>

Pyimgui 是一个基于 Cython 的 Python 绑定层，它将功能强大的用户界面库 Dear ImGui 无缝集成到 Python 环境中。它使 Python 开发人员能够轻松地创建交互式图形用户界面 (GUI)，同时充分利用 Dear ImGui 的丰富功能集。
