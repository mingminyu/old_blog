# Python-Barcode 教程

<show-structure depth="3"/>

前面介绍了生成二维码的方法，这次介绍一个快速生成条形码的模块。python-barcode 是一个用于生成条形码的Python库。它支持多种类型的条形码，包括CODE128、CODE39、EAN、UPC等，并允许用户自定义条形码的各种属性，如大小、颜色、字体等。
