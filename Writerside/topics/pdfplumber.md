# PDFPlumber 教程

<show-structure depth="3"/>

pdfplumber 是一个 Python 库，专为从 PDF 文件中提取文本和表格数据而设计。

与其他 PDF 处理库相比，pdfplumber 更注重保持页面上文本的视觉布局，这使得它在处理包含复杂布局或多列文本的 PDF 文件时表现更为出色。pdfplumber 依赖于 PDFMiner 来解析 PDF 文件，但提供了更友好和直观的 API。
