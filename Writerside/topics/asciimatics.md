# AsciiMatics 教程

<show-structure depth="3"/>

asciimatics 是一个 Python 库，用于创建跨平台的全屏显示终端应用程序。它能够生成 ASCII 艺术风格的图形，并且可以处理键盘输入和鼠标点击事件。asciimatics 适合于创建简单的游戏、演示、用户界面或者任何需要全屏终端图形的应用程序。
