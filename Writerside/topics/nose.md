# Nose 教程

<show-structure depth="3"/>

nose 是一个曾经流行的 Python 测试框架，它提供了对 Python 标准 unittest框架的扩展，使得测试变得更加灵活和强大。nose 支持更多的测试特性，如自动发现测试、灵活的测试运行器和插件系统，它在 Python 社区中被广泛用于简化测试过程和提高测试效率。
