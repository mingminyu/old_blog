# PyScript 教程

<show-structure depth="3"/>

pyscript 是一个允许用户使用 Python 语言直接在网页浏览器中创建和运行应用的框架。这个库的目标是让前端开发更加便利，同时使得Python 开发者能够更轻松地利用 Python 的强大功能来构建和分享网页应用。
