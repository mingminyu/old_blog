# FreezeGun 教程

<show-structure depth="3"/>

FreezeGun 是一个 Python 测试工具，用于模拟日期和时间，使得测试中的时间依赖性代码能够稳定地执行。它的主要作用是冻结时间，即让测试中的时间保持固定，不受系统当前时间的影响，从而使得测试结果可预测且稳定。
