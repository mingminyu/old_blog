# RDFLib 教程

<show-structure depth="3"/>

RDFLib是一个Python库，用于处理RDF（Resource Description Framework，资源描述框架）。

RDF是一种语义网标准，用于描述网络上的信息。RDFLib提供了一种方法来解析和创建RDF图形，以及查询和序列化它们。
