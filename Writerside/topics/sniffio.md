# Sniffio 教程

<show-structure depth="3"/>

Sniffio 是一个 Python 库，它提供了以与 asyncio 和 Trio 兼容的方式处理异步 I/O（输入/输出）的实用程序。它提供了一个一致的接口来检测和管理不同的事件循环，从而更容易编写无需修改即可与 asyncio 或 Trio 一起使用的代码。
