# MonkeyType 教程

<show-structure depth="3"/>


MonkeyType 是由 Instagram 开发的一个 Python 库，用于自动为 Python 代码生成类型注解。通过监控运行时的类型信息，MonkeyType 能够帮助开发者为现有的 Python 代码库增加类型提示，从而提高代码的可读性和健壮性。
