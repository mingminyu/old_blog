# Django-Vue-Admin 教程

<show-structure depth="3"/>

Django-Vue-Admin 是一款开源的开发平台，专为构建基于 RBAC 模型的权限控制的中小型企业级应用程序而设计。它采用先进的前后端分离架构，后端采用 Django 和 Django Rest Framework，前端采用 Vue 和 ElementUI，并支持通过 UniApp 和 uView 构建移动端应用（可发布为 H5 或小程序）。
