# Starlette 教程

<show-structure depth="3"/>

Starlette  是一个轻量级的  ASGI（异步服务器网关接口）框架，专为构建高性能的微服务而生。

不同于  FastAPI  的高级抽象，Starlette  提供了底层的  API  构建方式，允许开发者自由地定制每一个环节。
