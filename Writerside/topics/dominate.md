# Dominate 教程

<show-structure depth="3"/>

Dominate，顾名思义，它允许你通过  Python  来主导你的  HTML  文档。
由  Tom Flanagan  等人开发，这个库提供了一个优雅的  DOM API  来创建、操纵和渲染  HTML  文档。
与传统的编写  HTML  字符串方式相比，Dominate  让你可以用对象导向的方式来构建，从而在动态生成  HTML  时更为得心应手。
