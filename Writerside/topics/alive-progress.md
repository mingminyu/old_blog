# AliveProgress 教程

<show-structure depth="2"/>

alive-progress 是一个 Python 库，用于在控制台中创建活跃的进度条。它提供了丰富的配置选项，包括各种样式的进度条、进度块和旋转指示器。使用 alive-progress 可以轻松地为长时间运行的任务添加一个漂亮的进度条，提高用户体验。
